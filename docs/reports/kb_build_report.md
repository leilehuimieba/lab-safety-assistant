## 数据来源

### 1. 手工精标条目（KB-1001 ~ KB-1043）

共 43 条，分三批建立：

**第一批（KB-1001 ~ KB-1025，初版 25 条）**
面向高校化学、电气、通用实验室最常见问题，参考《高等学校实验室安全规范》（教科信厅函〔2023〕5号）等文件手工整理。

**第二批（KB-1026 ~ KB-1037，+12 条）**
新增覆盖此前空白的场景：

| ID 范围 | 主题 |
|---------|------|
| KB-1026 ~ 1027 | 辐射安全（准入要求、辐射污染应急） |
| KB-1028 | 激光安全操作规范 |
| KB-1029 | 液氮/低温安全与冻伤处置 |
| KB-1030 | 离心机安全使用 |
| KB-1031 | 高压灭菌锅操作规范 |
| KB-1032 | 灭火器类型选择与使用方法 |
| KB-1033 | 实验室应急疏散规程 |
| KB-1034 | 生物安全柜使用规范 |
| KB-1035 | 实验室安全等级分类制度 |
| KB-1036 | 危化品储存量限制规定 |
| KB-1037 | 化学品眼部灼伤应急处置 |

**第三批（KB-1038 ~ KB-1043，MSDS 专项 +6 条）**
针对实验室最常见危化品，参照 GB/T 16483 格式整理 MSDS 要点：

| ID | 化学品 | 主要风险 |
|----|--------|---------|
| KB-1038 | 丙酮（acetone） | 极易燃、挥发性、麻醉性 |
| KB-1039 | 甲醇（methanol） | 高毒、易燃、致盲风险 |
| KB-1040 | 氢氧化钠（NaOH） | 腐蚀性、强碱、放热溶解 |
| KB-1041 | 浓硫酸（H₂SO₄） | 强腐蚀、氧化性、稀释放热 |
| KB-1042 | 氯仿（三氯甲烷） | 潜在致癌（IARC 2B）、麻醉性 |
| KB-1043 | 甲苯（toluene） | 易燃、神经毒性、皮肤可吸收 |
