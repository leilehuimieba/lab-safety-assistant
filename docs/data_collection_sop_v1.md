# 实验室安全小助手 数据采集SOP v1

## 1. 目标与适用范围
本SOP用于指导“找数据同学”稳定收集可入库的数据，服务于以下三类功能：

- 页面展示用的公开材料与案例
- 智能体问答用的规范与制度依据
- 实验室安全小助手场景化问答（操作步骤、禁忌、应急）

本SOP适用于第一阶段（MVP到可演示版本），强调高质量、可追溯、可复核，不追求一次性“全网抓满”。

## 2. 数据采集目标（建议量化）
第一阶段建议目标（4到6周）：

- 文档源：60到100份高价值文档
- 可用知识条目（清洗后chunk）：800到1500条
- 核心高风险场景覆盖：至少20个
- 每个核心场景至少2个独立来源交叉验证

按类别建议占比：

- 校内制度与流程：30%
- 国家/行业规范：25%
- SOP与设备安全说明：20%
- MSDS/SDS：15%
- 应急预案与事故案例：10%

## 3. 必收数据类型与来源优先级
优先级从高到低：P0 > P1 > P2。P0必须先完成。

## 3.1 P0（必须）
- 校内实验室安全制度、准入制度、危化品管理制度、应急预案
- 校内实验室SOP（通风柜、危化品领用、废液处理、电气安全）
- 常用危化品SDS/MSDS（乙醇、丙酮、盐酸、硫酸、硝酸、氢氧化钠、甲醇等）

推荐来源：

- 本校实验室与设备管理部门、学院实验中心
- 校内课程平台或培训资料库（仅取可共享版本）
- 供应商官方SDS页面（Merck/Sigma、TCI、Fisher等）

## 3.2 P1（强烈建议）
- 国家标准/行业标准中的实验室相关条款
- 消防、电气、气瓶等实验室高风险相关规范
- 安全培训课件与考核题库（可公开或授权）

推荐来源：

- 国家标准全文公开系统（关键词检索“实验室 安全”“危化品”“通风柜”等）
- 应急管理部门与教育系统公开发布页
- 学校公开通知、实验室安全教育专题页

## 3.3 P2（可选扩展）
- 事故案例汇编（用于警示和解释，不作为唯一事实依据）
- 设备说明书（关键安全章节）
- 学术机构安全手册（用于补充）

## 4. 采集“准入标准”（收什么，不收什么）
文档准入（全部满足才入库候选）：

- 与实验室安全直接相关
- 来源可追溯（有机构/链接/版本信息）
- 内容具有操作性或约束性，不是纯宣传口号
- 时间有效（优先近5年，历史文档需注明版本）

不纳入（直接拒收）：

- 来源不明的二手转载
- 无法确认版权或授权范围的整本资料
- 与实验室安全弱相关的泛科普内容
- 重复文件（内容重复度高，仅保留最佳版本）

## 5. 采集格式要求（交付给清洗同学）
## 5.1 原始文件格式
- PDF、DOCX、PPTX、HTML链接（网页）
- 图片扫描件可收，但需在记录中标记“可能需要OCR”

## 5.2 文件命名规范
统一命名：

`来源机构_文档主题_版本或日期_语言.扩展名`

示例：

`WuyiU_实验室危化品管理制度_2025-09_zh-CN.pdf`
`Merck_Ethanol_SDS_2024-11_en.pdf`

## 5.3 元数据最小字段（必须填写）
采集同学提交时必须附一条记录，字段如下：

- `path`：文件相对路径
- `source_title`：文档标题
- `source_org`：发布机构
- `category`：大类（制度/SOP/SDS/标准/应急/设备）
- `subcategory`：子类
- `lab_type`：化学/生物/电气/通用
- `risk_level`：1到5
- `hazard_types`：如化学;电气;高压气体
- `language`：zh-CN/en
- `question_hint`：该文档能回答的核心问题
- `reviewer`：采集人

建议直接填写到仓库模板：

- `data_sources/document_manifest.csv`
- 或先用模板 `data_sources/document_manifest_template.csv` 再合并

## 6. 采集同学每日交付清单
每日最小交付：

- 新增文档 5到10 份（P0/P1优先）
- 每份文档一条元数据记录
- 一个简短日志（今天新增、问题、明天计划）

交付形式：

- 提交到个人分支：`feat/data-source-<name>-<date>`
- 发起PR到 `main`（或项目约定的集成分支）

## 7. 质量门槛（采集阶段）
每周汇总时，采集数据应达到：

- P0来源占比不低于60%
- 元数据完整率 >= 98%
- 重复文件率 <= 10%
- 来源可追溯率 = 100%

## 8. 团队分工建议
- 数据采集同学：负责找资料、命名、填元数据
- 数据清洗同学：负责解析、去噪、规则修正、产出CSV
- 评测同学：负责跑评测集与验收报告
- 项目负责人：合并PR、把控优先级和验收

## 9. 推荐搜索关键词（给采集同学直接用）
- `实验室 安全 管理 制度 pdf`
- `实验室 通风柜 使用 规范`
- `危化品 领用 储存 处置 规范`
- `高校 实验室 应急 预案`
- `实验室 电气 安全 操作 规程`
- `化学品 SDS 乙醇 丙酮 盐酸`
- `实验室 气瓶 使用 安全`
