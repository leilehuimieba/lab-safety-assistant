# 实验室安全小助手

一个面向高校实验室场景的实验室安全问答与辅助决策项目。

本仓库主要保留项目申报、知识库样例、规则库、评测集与阶段性调参文档，适合作为项目展示、课程申报、阶段汇报和后续二次开发的基础材料。

## 仓库内容

- `实验室安全小助手_项目申报立项书_可提交版.docx`：项目申报书提交版
- `实验室安全小助手_项目申报立项书_可提交版.md`：项目申报书 Markdown 版
- `knowledge_base_curated.csv`：清洗后的核心知识库样例
- `safety_rules.yaml`：规则库与应急响应规则
- `eval_set_v1.csv`：项目评测集
- `eval_criteria.md`：验收与评测指标
- `retrieval_tuning_report.md`：检索与召回调参记录

## 项目定位

项目目标是构建一个“实验室安全小助手”，能够围绕实验室安全制度、危险操作提醒、应急处置流程、实验前检查和常见合规问题提供辅助问答能力。

当前版本强调以下几个方向：

- 基于知识库的实验室安全问答
- 面向高风险场景的规则化安全约束
- 面向项目验收的评测集与指标设计
- 面向 Dify 的快速原型搭建路径

## 当前状态

当前仓库包含的是项目核心资料与可公开的阶段成果，不包含以下运行时内容：

- Dify 运行数据卷
- 本地数据库与上传文件
- API 密钥与私有配置
- 临时调试输出与辅助同步脚本
